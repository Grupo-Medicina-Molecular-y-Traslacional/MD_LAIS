RELEASE NOTES

MD-LAIs (v1.0) is the first public release of the application.

This version includes:

1) Several sets of aggregation operators (i.e., Non-classical (Norms, Means, Statistical) and Classical (Information Theory Autocorrelation, Kier-Hall, Radial Distribution Function, MoRSE and so on).

2) Thirty groups to compute local molecular descriptors. 8400 k-mers as groups for local calculations.

3) Generalized slide-window-based amino acid-level descriptors

4) Thirty-nine amino acid properties (chemical-structural, topological/hydrophobic-scales, MD-LOVIS and QUBILS-MIDAS indices)

5) Amino acid fuzzy weights according to the distance to a point in the sequence.
